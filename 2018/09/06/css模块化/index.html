css模块化

2018-09-06

这是今天发现的一个新东西，提到vue中的css模块化，首先我想到的是style标签中的scoped属性,但它只是为了避免当前组件内的样式修改其他地方的样式。

也许外部的同名样式还是可以修改当前组件内的样式，比如添加!important。

scoped是在标签上添加一个唯一的data属性，同时在css选择器上加上相应的data标签选择器，其实是增加了选择器的权重的。

但css module，它是把标签和css选择器的class名称直接替换成一串唯一的字符串。外部样式就很难去影响组件内的样式，本身样式的权重也不会受到影响。
